# 🏀 NBA Team Performance Dashboard

An interactive Streamlit dashboard for exploring NBA team and player performance during the 2024–2025 NBA regular season. Select a team and season to view team statistics, starting lineups, positional analysis, and player-to-team performance comparisons through interactive visualizations.

[![Open in Streamlit](https://static.streamlit.io/badges/streamlit_badge_black_white.svg)](https://gdp-dashboard-template.streamlit.app/)


### Features

- 🏀 Filter by NBA team and season
- 🎨 Dynamic team branding with official colors and logos
- 📊 League and team average statistics
- 🥇 Starting Five player profiles with headshots and key stats
- 🏠 Home vs. Away win rate donut chart
- 📈 Position-based performance analysis using Altair
- 🕸️ Interactive Player vs. Team radar chart comparison

### Dataset

The dashboard uses player-level game statistics from the **2024–2025 NBA Regular Season**. The dataset includes:

- Player information and headshots
- Team information and logos
- Game results
- Shooting statistics
- Rebounds, assists, steals, and blocks
- Team scores
- Official team colors

### Technologies

- Streamlit
- Python
- Pandas
- Altair
- Plotly

### How to run it on your own machine

1. Install the requirements

   ```
   $ pip install -r requirements.txt
   ```

2. Run the app

   ```
   $ streamlit run streamlit_app.py
   ```
